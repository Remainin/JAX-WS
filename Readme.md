# 哈工大计算机学院-服务计算实验

实现代码为Java，代码下载后直接可用，客户端的UI界面如下图：<br>

用JAX-WS模拟实现某公司的产品的信息管理服务，其中至少包括：产品列表查询、某种产品的详情查询（含描述获取，库存查询，价格查询等）、对某种产品的下单、收货确认回执。
